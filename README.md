.TH C - Simple Shell

.PP

.SH DESCRIPTION

Simple Shell is a simple UNIX command line interpreter written in C.

The project recreates a basic version of the UNIX shell and demonstrates
how command execution works internally in operating systems.

The shell reads commands entered by the user, processes them,
and executes programs using system calls such as
.B fork
and
.B execve .

.PP

This shell supports interactive mode, command execution,
argument handling, PATH resolution, and basic built-in commands.

.PP

The purpose of this project is to understand process creation,
system programming, memory management, and how shells communicate
with the operating system.

.PP

This project was developed as part of the Holberton School simple shell curriculum.
